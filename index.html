Nick Rutowski

Web Developer - Geek - Maker of Things

Hi! I'm Nick Rutowski, some people know me as Badsauce or Codesauce (most of those people are me).

I'm a big fan of web design, I've dedicated a lot of time to creating enterprise websites using ASP.NET MVC . I've also spent some time getting paid to build sites using CakePHP on the LAMP stack - Those were dark days, I try not to think about them.

In my free time I have messed around with a slew of other web technologies, on the MVC front I've dabbled in Django and recently started exploring the Node.js space with Sails.js (This site is built on sails =O). On the subject of javascript, I've done some data visualization with d3.js and loved every minute of it.

I write a lot of C++ and Python for scholarly pursuits like http routing and learning how pointers work.

Places you can probably find me: Email , Github , or Twitter .
